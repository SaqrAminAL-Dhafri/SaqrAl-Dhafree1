مقارنة شاملة بين أشهر محركات قوالب جانغو

محرك القوالب هو المسؤول عن عرض البيانات الديناميكية داخل ملفات HTML. نستعرض هنا مقارنة بين أربعة من أشهر هذه المحركات.

1. Django Template Language (DTL)

هو المحرك الرسمي والمدمج في جانغو. فلسفته الأساسية هي فرض الفصل الصارم بين منطق العرض ومنطق العمل.

  • نقاط القوة: تكامل مثالي، سهولة التعلم، أمان عالي.
  • نقاط الضعف: مرونة محدودة، الفشل الصامت (لا يطلق أخطاء للمتغيرات غير المعرفة).

2. Jinja2

البديل الأكثر شعبية لـ DTL، مستوحى منه لكن مع مرونة أكبر وأداء أسرع.

  • نقاط القوة: أداء سريع، مرونة عالية، تصحيح أخطاء أفضل.
  • نقاط الضعف: يتطلب إعداداً إضافياً.

3. Mako

محرك قوالب عالي الأداء معروف بسرعته الفائقة وفلسفته التي تمنح المطور الحرية الكاملة لكتابة كود بايثون مباشرة داخل القوالب.

  • نقاط القوة: سرعة فائقة، قوة مطلقة ومرونة كاملة.
  • نقاط الضعف: خطر خلط المنطق بالعرض، أقل شيوعاً في مجتمع جانغو.

ملخص شامل لفلاتر Django Template Language (DTL)

الفلاتر هي أدوات بسيطة تستخدم لتعديل شكل عرض المتغيرات في القالب. الصيغة العامة لاستخدامها هي {{ variable|filter_name }}.

1. فلاتر التعامل مع النصوص

  • upper: لتحويل كل الحروف إلى حروف كبيرة. مثال: {{ "hello"|upper }} ← HELLO
  • lower: لتحويل كل الحروف إلى حروف صغيرة. مثال: {{ "HELLO"|lower }} ← hello
  • title: لتحويل الحرف الأول من كل كلمة إلى حرف كبير. مثال: {{ "hello world"|title }} ← Hello World
  • truncatewords: لقطع النص بعد عدد معين من الكلمات. مثال: {{ "هذه جملة طويلة جدا"|truncatewords:2 }} ← هذه جملة...
  • slugify: لتحويل النص إلى صيغة مناسبة للروابط (URL). مثال: {{ "My New Post!"|slugify }} ← my-new-post

2. فلاتر التعامل مع القوائم

  • length: لإرجاع عدد العناصر في القائمة.
  • first: لإرجاع العنصر الأول في القائمة.
  • last: لإرجاع العنصر الأخير في القائمة.
  • join: لربط عناصر القائمة بنص معين. مثال: {{ ["a", "b"]|join:" - " }} ← a - b

3. فلاتر التعامل مع التواريخ والأوقات

  • date: لتنسيق عرض التاريخ والوقت. مثال: {{ my_date|date:"Y-m-d" }}
  • timesince: لعرض المدة التي مضت منذ تاريخ معين. مثال: "منذ 5 ساعات"

4. فلاتر أخرى مهمة

  • default: لتحديد قيمة افتراضية إذا كان المتغير فارغاً. مثال: {{ my_var|default:"لا يوجد" }}
  • filesizeformat: لتحويل رقم (بايت) إلى صيغة سهلة القراءة. مثال: {{ 1234567|filesizeformat }} ← 1.2 MB
  • safe: لوضع علامة على نص HTML بأنه "آمن". (استخدمه بحذر!)
